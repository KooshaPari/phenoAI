# PhenoAI - AI/LLM Workspace

AI-related crates for the Pheno ecosystem.

## Crates

| Crate | Description |
|-------|-------------|
| `llm-router` | Multi-provider LLM routing |
| `mcp-server` | MCP protocol server |
| `pheno-embedding` | Embedding pipeline |

## Usage

```toml
[dependencies]
llm-router = { path = "../phenoAI/crates/llm-router" }
mcp-server = { path = "../phenoAI/crates/mcp-server" }
```
